## Dados utilizados

Esta pasta contém os arquivos de dados utilizados no projeto, obtidos a partir de fontes públicas oficiais.

### Arquivos

- bacen_concessoes_financiamento_imobiliario_pf_tratado.csv  
  Série temporal mensal (2015–2025) do volume de concessões de financiamento imobiliário para pessoas físicas, expressa em milhões de reais.

### Fonte

- Banco Central do Brasil (BACEN) — Sistema Gerenciador de Séries Temporais (SGS)

### Observações

- Os valores monetários estão mantidos em milhões de reais.
- Os dados passaram por padronização de datas e validação antes da análise.
